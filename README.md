## Description
A Minecraft minigame plugin for PaperMC where you fight epic duels against other players in a vast arena!
In stage one, your job is to collect resources found in crates, but don't get caught by your nemesis, or the game will find an early ending.
If you survive long enough, it will be time for a deathmatch, where there will be no escape!

Winner takes all!
